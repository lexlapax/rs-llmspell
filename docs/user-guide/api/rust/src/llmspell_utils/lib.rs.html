4//! # `LLMSpell` Utilities
5//!
6//! This crate provides shared utility functions and helpers used across the `LLMSpell` framework.
7//!
8//! ## Features
9//!
10//! - **Async Utilities**: Helpers for working with async operations, timeouts, and concurrency
11//! - **File Utilities**: Cross-platform file operations and path manipulation
12//! - **String Utilities**: String manipulation, formatting, and validation helpers
13//! - **System Info**: System information gathering and environment utilities
14//! - **Error Builders**: Convenient error construction helpers
15//! - **ID Generator**: UUID-based ID generation for components
16//! - **Serialization**: Common serialization/deserialization utilities
17//!
18//! ## Usage
19//!
20//! ```rust,ignore
21//! use llmspell_utils::{async_utils, file_utils, id_generator};
22//! use std::time::Duration;
23//!
24//! // Generate a unique component ID
25//! let id = id_generator::generate_component_id("agent");
26//!
27//! // Use async utilities for timeout operations (when implemented)
28//! let result = async_utils::timeout(Duration::from_secs(30), async {
29//!     // Your async operation
30//! }).await?;
31//! ```
