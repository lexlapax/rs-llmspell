16//! # Session and Artifact Management
17//!
18//! This crate provides comprehensive session management capabilities for rs-llmspell,
19//! including:
20//!
21//! - Session lifecycle management (create, save, restore, replay)
22//! - Artifact storage with content-addressed deduplication
23//! - Hook integration for session events
24//! - Event correlation across session activities
25//! - Lua bridge for script access to sessions
26//!
27//! ## Architecture
28//!
29//! The crate is organized into the following modules:
30//!
31//! - `manager`: Core session management orchestration
32//! - `session`: Session types and state management
33//! - `artifact`: Artifact storage and retrieval
34//! - `replay`: Session replay engine
35//! - `bridge`: Script language integration (Lua)
36//! - `error`: Error types and handling
37//! - `types`: Core type definitions
38
