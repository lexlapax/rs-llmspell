Function append_file

pub fn append_file(path: &Path, data: &[u8]) -> Result<()>
Expand description

Append data to a file

This function appends data to an existing file or creates it if it doesn’t exist. The parent directory is created if necessary.

§Examples

use llmspell_utils::file_utils::append_file;
use std::path::Path;

append_file(Path::new("/tmp/log.txt"), b"\nNew log entry")?;

§Errors

Returns an error if:

  • Parent directory cannot be created
  • Write permissions are denied
  • I/O error occurs
